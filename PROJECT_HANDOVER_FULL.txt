6. Database Design / 数据库设计
============================================================

Current main database tables:

- users
- books
- reading_lists
- reading_list_items
- reviews

Design intention:

- users store authentication-related user records
- books store locally cached books that were added into lists
- reading_lists store user-owned list containers
- reading_list_items store list-book relationships and status
- reviews store user reviews on locally cached books

Key database relationships:

- one user -> many reading lists
- one reading list -> many reading list items
- one user -> many reviews
- one book -> many reviews

Important design note:

The external source (Open Library) is used for live book search and details,
while selected books are cached locally for analytics and recommendation logic.
